# 一、工作场景引发的思考

我的本职工作中会传达一些需求。所以在梳理我的问题时就迁移思考了实际的工作经历。

每一个产品经理都有将需求完整的描述给项目组其他成员的经历。而在描述需求的时候，最简单的办法之一就是引用场景。

那么在提示词中，是不是添加具体的场景，可以让 AI 更好的理解我们的描述，更好的完成我交给它的任务呢？
